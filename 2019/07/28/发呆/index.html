发呆

—-我
站在阳台
向下看
风抚着白千层的枝条
摇曳
阳光洒在阳台的仙人掌上
金色的绒毛，像极了

发呆
只有耳边的旋律
冬日的阳光
慵懒着，宠溺
不想动
只愿被光融化了
飘散到小草 大树 行人身上
留下一丝温热
那就好了

这是在刚入大学时候，在宿舍写得诗，是的，这是一首诗，是我生活的诗，它看起来不美，
但是足够敲打我的心弦。有时候，真感激当初的自己，在奔涌向前的时光流水里，留下了一
只小纸船，让我能够在今后的某一天，回头时，能窥见自己当初的样貌。以前看着真傻，看
着真好。
